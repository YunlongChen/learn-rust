# 域名管理GUI - 项目路线图

## 迭代原则

1. **核心功能优先**：先实现基础域名管理能力
2. **用户体验驱动**：逐步优化界面和交互
3. **渐进式扩展**：从单云支持到多云适配
4. **质量保障**：每个版本包含测试和文档

## 依赖技术栈

- **前端框架**: Tauri (Rust + WebView)
- **状态管理**: Yewdux
- **云SDK**: 阿里云OpenAPI, Cloudflare Rust SDK
- **测试框架**: Playwright (E2E), Criterion (性能)
