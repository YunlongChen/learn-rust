STUN 信息

GET /api/v1/stun/info

获取 STUN/TURN 服务器信息，包含 stun_addr、turn_addr、public_ip

响应示例

{
  "code": 0,
  "message": "success",
  "data": {
    "stun_addr": "127.0.0.1:3478",
    "turn_addr": "127.0.0.1:3478",
    "public_ip": "127.0.0.1"
  }
}

Agent 管理

POST /api/v1/agent/register

注册 Agent

请求参数（JSON body）

  • agent_id (UUID) — Agent 唯一标识
  • name (string) — Agent 名称
  • public_addr (string) optional — 公网地址
  • nat_type (string) — NAT 类型

响应示例

{
  "code": 0,
  "message": "success",
  "data": {
    "relay_token": "uuid-here"
  }
}

  }
}
